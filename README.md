# Run Manually

MongoDB should be running on `127.0.0.1:27017`


## Client


`$ cd client`


`$ npm i`


`$ npm run dev`


## Server


`$ cd server`


`$ npm i`


`$ npm run start`


Then visit `http://localhost:3000`


# Run with Docker (includes mongodb)


In the root of the repository, run


`docker-compose up --build`


Then visit `http://localhost:3000`
